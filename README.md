2. # 🚀 AI Data Center Control Tower

An AI-powered real-time dashboard for energy optimization, risk management, and predictive analytics in data center operations.

## 🔥 Features

- Real-time energy monitoring
- Machine learning predictions (Random Forest)
- 24-hour energy forecasting
- Anomaly detection (Isolation Forest)
- Digital twin simulation (what-if analysis)
- Cost optimization (€ insights)
- Risk & disaster management triggers
- Executive storytelling dashboard
- Multi-channel alerts (Email + Teams)

## 🧠 Business Value

- Reduce energy consumption by 15–25%
- Improve operational efficiency
- Enable proactive risk mitigation
- Provide executive-level insights

## 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- Plotly
- Pandas / NumPy
